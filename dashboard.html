Create a GitHub token
The token lets the server push your usage data to a Gist — it creates the Gist automatically.

1. Open github.com/settings/tokens/new
2. Name it anything (e.g. claude-usage)
3. Select scope: gist only
4. Click Generate — copy the token (you only see it once)
4
Enter your token
Saved to a local .env file. Never shared. A Gist is created for you automatically.
